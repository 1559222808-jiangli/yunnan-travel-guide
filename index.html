📅 每日详细行程

🌿 3月24日 | 昆明抵达日（精简不绕路）

方案A：昆明留宿（轻松版）

16:10 长水机场 → 地铁 → 昆明站附近酒店
18:00 南屏街 → 金马碧鸡坊 → 昆明老街（吃过桥米线/汽锅鸡）
22:00 休息，次日早班动车去大理

方案B：当晚去大理（省时版）

16:10 长水机场 → 地铁 → 昆明站 → 晚班动车 → 大理站 → 大理古城民宿
21:30 入住休息（直接跳过昆明绕路，留更多时间给洱海）

🌊 3月25日 | 大理：古城初体验+洱海西线日落

08:00 大理站 → 大理古城（办理入住）
09:00 大理古城（白天逛五华楼/人民路，留晚上继续逛）
14:00 古城 → 租电动车/吉普车 → 磻溪S湾 → 生态廊道 → 才村码头
18:30 才村码头 → 兴盛大桥 洱海日落（蓝调时刻绝美）
20:00 古城夜景（洋人街/酒吧街，晚上逛更有氛围）

🌅 3月26日 | 大理：洱海日出+环海东路深度游（核心日）

06:30 古城 → 打车 → 龙龛码头 洱海日出（3月约7:10，提前30分钟占位）
08:30 龙龛码头 → 喜洲古镇（四方街/转角楼，吃现烤喜洲粑粑）
12:00 喜洲 → 双廊古镇（午餐：洱海砂锅鱼）
14:00 双廊 → 玉几岛 → 南诏风情岛（可选游船，感受碧波）
17:30 双廊 → 挖色码头 洱海日落（苍山+金色湖面同框）
19:30 双廊 → 包车 → 大理古城

🏯 3月27日 | 大理：三塔+苍山远观+古城夜逛

09:00 古城 → 崇圣寺三塔（拍三塔倒影，门票75元）
12:00 三塔 → 苍山脚下 远观苍山（不必上山，节省体力）
14:00 返回古城 → 床单厂艺术区 → 洱海门
19:00 古城夜景（人民路/五华楼俯瞰，补逛第一天没走完的角落）

🍃 3月28日 | 大理→丽江：有风的地方+转场

16:00 古城 → 大理站 → 动车 → 丽江站 → 丽江古城民宿
20:00 丽江古城夜景（四方街/酒吧街，吃腊排骨火锅）

🏔️ 3月29日 | 丽江：日照金山备选+古城+束河

19:00 束河 → 丽江古城（吃野生菌火锅）

🏔️ 3月30日 | 丽江：日照金山+玉龙雪山一日游（核心日）

05:30 古城 → 包车 → 甘海子 日照金山最佳（3月约6:50，无遮挡）

12:00 冰川公园 → 蓝月谷 → 云杉坪（“小九寨”蓝月谷必拍）
16:00 景区 → 丽江古城
19:00 古城夜景（可选《丽水金沙》演出，体验纳西文化）

🏡 3月31日 | 丽江：白沙古镇+拉市海

07:00 古城 → 白沙古镇 日照金山备选（民宿露台看雪山，商业化最弱）

13:00 白沙 → 玉水寨（纳西文化发源地，神泉瀑布）
16:00 玉水寨 → 拉市海（骑马+划船，看湿地日落）
19:30 拉市海 → 丽江古城

✈️ 4月1日 | 丽江：古城收尾+返程

✨ 核心景点打卡指南、实用贴士、避坑提醒

🌅 洱海日出

  • 最佳机位：龙龛码头（红杉林+水中栈道）
  • 时间：3月约7:10，提前30分钟到占位
  • 穿搭：浅色衣服+围巾，出片率拉满

🌇 洱海日落

  • 最佳机位：兴盛大桥（蓝调时刻）、挖色码头（苍山+湖面）
  • 时间：3月约19:00，提前1小时到

🌄 日照金山

  • 最佳机位：甘海子（震撼无遮挡）> 黑龙潭（倒影经典）> 白沙古镇（人文+自然）
  • 时间：3月约6:50，提前40分钟到
  • 关键：必须晴天！提前2天看天气预报，预留2天观赏机会

👔 穿搭建议

  • 洋葱式穿搭：内搭+薄外套+防风冲锋衣（早晚温差10-15℃）
  • 必备：墨镜、帽子、SPF50+防晒霜、润唇膏（高原干燥）
  • 玉龙雪山：加一件厚外套/羽绒服（可租，50元/件）

🩺 高反预防

  • 逐步适应海拔：昆明(1900m)→大理(2000m)→丽江(2400m)→玉龙雪山(4680m)
  • 玉龙雪山：备1-2瓶氧气瓶（提前买30元/瓶，景区80元/瓶），不跑跳、多喝水
  • 可提前3天吃红景天（辅助预防，非特效药）

🚗 交通避坑

  • 大理租电动车：选正规店铺，戴头盔，确认电池续航（环海需满电）
  • 丽江包车：拒绝低价黑车，民宿推荐的师傅更靠谱
  • 景区电瓶车：玉龙雪山蓝月谷电瓶车必坐（步行需1小时）

🍜 美食避坑

  • 避开古城门口拉客餐厅，走进去50米以上，选本地人常吃的店
  • 大理必吃：喜洲粑粑（现烤咸口）、白族酸辣鱼、乳扇（街边现烤）
  • 丽江必吃：腊排骨火锅、纳西烤肉、鸡豆凉粉（忠义市场附近）

⚠️ 其他提醒

  • 日照金山/洱海日出日落：提前2天查天气预报，预留2天观赏机会
  • 伴手礼：鲜花饼（嘉华/潘祥记）、牦牛肉干（正规超市买）
  • 现金：备少量（部分小店/景区厕所需现金）
